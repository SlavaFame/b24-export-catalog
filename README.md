# Bitrix Catalog Section Exporter

Скрипт для экспорта раздела каталога со всеми вложенными разделами и товарами, для отправки данных через POST-запросы.

## Описание

Скрипт выполняет экспорт структуры каталога (разделы, товары, свойства, цены). 
Используются только методы ядра Битрикса, без REST API. 
Данные разбиваются на пакеты для оптимизации загрузки.
